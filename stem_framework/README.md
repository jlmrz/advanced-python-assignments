## Stem framework

Command for deployment:

Create and activate virtual environment
```bash
python3 -m venv .venv
source .venv/bin/activate
```
Go to stem_framework and install requirements:
```bash
cd ./stem_framework
python3 -m pip install -r requirements.txt
pip install -e .
```
